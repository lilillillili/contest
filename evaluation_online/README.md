# 온라인 서류평가 심사자료 만들기

## 🚀 설명
- 각 팀에서 제출한 서류를 구글 드라이브에 업로드 후 링크 일괄 다운로드하여 메모장에 저장(linklist.txt)
- 구글링크 저장한 메모장(101행)과 심사표 템플릿 엑셀파일(104행) 준비
- 엑셀 템플릿 준비 및 메모장, 엑셀, 파이썬 파일 함께 있는 파일 경로 정확히 지정(98행)
- 구글 링크의 파일명에서 _(언더바)로 구분된 팀명 추출, 템플릿 엑셀 D열의 팀명과 매치 
- 엑셀 템플릿의 8행부터 마지막 행까지 순회, F열에 링크 추가(링크명:팀명 심사파일)한 결과파일 출력

## 🛠️ 기술 스택
- Python 3.7

## 🎯 설치 및 실행 방법
1. Python 라이브러리 설치:
   `pip install requests pandas openpyxl`
2. 메모장, 템플릿 엑셀파일 준비:
   `linklist.txt`
   `2025년 대회 예선 심사표.xlsx`
3. 파이썬 실행:
   `evaluate_excel.py`
4. 엑셀파일 결과 출력:
   `2025년 대회 예선 심사표_결과.xlsx`
